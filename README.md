# mindforgeai-exam-2-2026CS018
**Objective**

To analyze student performance using Python and Pandas by calculating readiness scores, classifying students into performance bands, and generating useful analytics.

**Tools Used**
- Python
- Google Colab
- Pandas
- GitHub

**How to Run**

1. Clone or download the repository.
2. Open the notebook in Jupyter Notebook or Google Colab.
3. Ensure the CSV file is available in the data/ folder.
4. Run all notebook cells sequentially.
5. Result Summary
6. Loaded and validated the student dataset.
7. Calculated readiness scores using attendance, diary entries, tasks, assignment, and lab scores.
8. Classified students into Support, Developing, Strong, and Excellent categories.
9. Generated department-wise averages, identified the top performer, and listed students needing academic support.
